
# ⚛ Blog Pessoal Frontend com React
Este é um blog pessoal desenvolvido para documentar aprendizados em tecnologia, utilizando React, Vite e TypeScript.

## 🚀 Tecnologias Utilizadas

* [React](https://react.dev/)
* [Vite](https://vitejs.dev/)
* [TypeScript](https://www.typescriptlang.org/)

## 📦 Como rodar o projeto localmente

1. **Clone o repositório:**
```bash
git clone https://github.com/Eliane-orlandin/blog-pessoal-frontend.git

```


2. **Entre na pasta:**
```bash
cd blogpessoal

```


3. **Instale as dependências:**
```bash
npm install
# ou yarn install

```


4. **Inicie o servidor de desenvolvimento:**
```bash
npm run dev
# ou yarn dev

```


5. **Acesse no navegador:**
O projeto estará rodando em `http://localhost:5173`.

## 🛠 Comandos Disponíveis

* `npm run dev`: Inicia o ambiente de desenvolvimento com Hot Module Replacement (HMR).
* `npm run build`: Compila o código para produção na pasta `dist`.
* `npm run preview`: Visualiza o build de produção localmente.
